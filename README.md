Web-service "BillBoard" as a pet-project. You can register, push your advertisement, comment it and find potential buyer. Include admin-site functional and user's profile.

Were used in development:
- MVC patern;
- ORM technology;
- Bootstrap4;
- REST API;
- PostreSQL.
